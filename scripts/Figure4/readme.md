# scripts/Figure4/ — Global Secondary Structure Heatmap & Allosteric Switches

## Purpose
Produces the global secondary structure heatmap (ordered-fraction across 1368 residues × 7 states) and identifies/plots residues that undergo large secondary structure shifts — the "allosteric switches".

## Scripts

| Script | Description |
|--------|-------------|
| `plot_ss_heatmap.py` | Reads the raw per-residue ordered-fraction data and generates the full-residue heatmap with domain boundary annotations. |
| `classify_switches_final.py` | Identifies residues with ordered-fraction shift ≥ 40 % across states; classifies them as "Folder" (disorder→order) or "Unfolder" (order→disorder); writes `Classified_Switches.csv`. |
| `plot_ss_distribution_final.py` | Generates distribution and per-domain summary bar plots of the classified switch residues. |

## Inputs

| File | Description |
|------|-------------|
| `raw_data/Global_Secondary_Structure/Global_SS_RawData.csv` | Per-residue ordered-fraction (%) for all 7 states |
| `data/Secondary_Structure_Shift/All_Residues_SS_Shift.csv` | Pre-computed shift statistics (generated by `classify_switches_final.py`) |

## Outputs

| File | Description |
|------|-------------|
| `figures/Figure4/Global_SS_Heatmap_final.png` | Full-residue secondary structure heatmap |
| `figures/Figure4/SS_Shift_Distribution.png` | Distribution of max shift magnitude |
| `figures/Figure4/Domain_Switch_Trends.png` | Per-domain Folder/Unfolder counts |
| `data/Secondary_Structure_Shift/Classified_Switches.csv` | Table of switch residues |

## Execution

```bash
# All three scripts read "Global_SS_RawData.csv" or their outputs as bare filenames;
# run them from raw_data/Global_Secondary_Structure/ where the input CSV lives.
cd raw_data/Global_Secondary_Structure

# Step 1: generate heatmap and compute All_Residues_SS_Shift.csv
python ../../scripts/Figure4/plot_ss_heatmap.py
python ../../scripts/Figure4/classify_switches_final.py   # reads Global_SS_RawData.csv, writes All_Residues_SS_Shift.csv

# Step 2: generate distribution plots (reads All_Residues_SS_Shift.csv)
python ../../scripts/Figure4/plot_ss_distribution_final.py
```

> **Note:** Output CSVs (`All_Residues_SS_Shift.csv`, `Classified_Switches.csv`) are written to the working directory. Move them to `data/Secondary_Structure_Shift/` after the run, or update the `OUTPUT_CSV` path variables at the top of each script.
