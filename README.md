## Folder Structure

| Folder | Description |
|--------|-------------|
| `data/raw/` | Original, unmodified data (read-only) |
| `data/processed/` | Cleaned data ready for analysis |
| `scripts/` | Analysis scripts (numbered by execution order) |
| `outputs/figures/` | Generated charts and visualizations |
| `outputs/tables/` | Summary tables and results |
| `docs/` | Methodology, codebook, and documentation |
| `references/` | Related literature and reference materials |

## Key Files

| File | Purpose |
|------|---------|
| `requirements.txt` | Python package dependencies |
| `.gitignore` | Files excluded from version control |
| `LICENSE` | Usage and redistribution terms |
