# figures/

This folder stores all **publication-quality output figures** (PNG, 600 dpi) generated by the scripts in `scripts/`.

## Sub-folders

### `Figure3/`
| File | Description |
|------|-------------|
| `Fig_Mechanistic_Domain_Dynamics_Clean.png` | Multi-panel plot of domain-median RMSF across R-loop states, grouped into "Rigid Scaffold", "Sensors and Transducers", and "Catalytic Payload" (HNH). Generated by `scripts/Figure3/Fig_Mechanistic_Domain_Dynamics_median.py`. |

### `Figure4/`
| File | Description |
|------|-------------|
| `Global_SS_Heatmap_final.png` | Heatmap of ordered-structure fraction (%) for all 1368 SpCas9 residues across 7 R-loop states, with domain annotations. |
| `SS_Shift_Distribution.png` | Distribution of maximum secondary-structure shift magnitude across residues. |
| `Domain_Switch_Trends.png` | Per-domain count of allosteric switch residues classified as Folders vs Unfolders. |

### `Figure5/`
| File | Description |
|------|-------------|
| `SB_Macroscopic_Domain_Network_Final.png` | Inter-domain salt-bridge interaction network diagram across R-loop states. |
| `SB_B_Net_Flux.png` | Net flux of salt-bridge interactions (appearing minus disappearing) at each state transition. |

### `Figure6/`
| File | Description |
|------|-------------|
| `GCCM_Compact_Summary.png` | Compact summary of GCCM-derived network properties (edge density, key hub metrics) across states. |
| `Path_Impedance_Groups.png` | Optimal allosteric path impedance for three functional residue groups across seven R-loop states. |

### `supplenmentary/`
| File | Description |
|------|-------------|
| `CB_VESM_DMS.png` | Integrated CB / VESM / MD percentile-rank scatter and top-30 stacked bar (Figure S9). Generated by `scripts/supplementary/AI-dms-validation/plot_integrated_rank.py`. |
| `CB_Directionality.png` | State-bias analysis of ProteinMPNN conditional probabilities across R-loop transitions (Figure S10). Generated by `scripts/supplementary/AI-dms-validation/plot_CB_state_bias_2rows.py`. |

---

> Figure 2 output (porcupine plot) is saved directly to the script working directory and is not stored here.  
> Figure 7 output (`CB_VESM_Discordance.png/.pdf`) is saved to the `scripts/Figure7/` working directory.  
> CB × VESM × DMS output (`Figure_CB_VESM_DMS.png/.pdf`) is saved to the `data/AI-validation-[CB,VESM]/CB_VESM_discordance/` working directory.
